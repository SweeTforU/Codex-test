# Codex-test: Movies Rating Fairness Experiment

本仓库用于测试 Codex 的编程与协作流程，当前聚焦于电影评分网站公平性实验。

## 分支说明
- `main`：主干分支。
- `topic/movies-test`：电影评分网站实验分支，用于验证更公平、透明、可追溯的评分机制。

## 当前实现
- `index.html`：前端 HTML 原型页面。
  - 支持添加电影（名称、年份、简介）。
  - 支持多维度打分（剧情 / 画面 / 表演）。
  - 评分聚合规则：`剧情*0.4 + 画面*0.3 + 表演*0.3`。
  - 使用 `localStorage` 持久化电影与评分数据。

## 项目目标（movies-test）
- 构建一个面向电影评分的站点原型。
- 聚焦评分公正性（反刷分、样本质量、评分可解释性）。
- 在后续迭代中增加防滥用机制与可信评分策略。
