# Codex-test: Movies Rating Fairness Experiment

本仓库用于测试 Codex 编程流程，当前在 `topic/movies-test` 分支实现电影评分公平性原型。

## 当前能力（已实现）
- 单页 HTML 原型：`index.html`
- 电影录入：名称 / 年份 / 简介
- 多维评分：剧情 / 画面 / 表演
- 双评分模型：
  - 基准分：`剧情*0.4 + 画面*0.3 + 表演*0.3`
  - 公平分：按“评分可信度权重”计算加权均值
- 公平机制：
  - 评分者 ID 维度频控（同用户同电影 60 秒冷却）
  - 短评缺失与维度异常差异的权重惩罚
- 本地持久化：`localStorage`

## 分支说明
- `main`：主干分支。
- `topic/movies-test`：电影评分公平性实验分支（本原型所在）。
